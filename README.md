## About The Project

This is my personal developer portfolio designed to present my technical skills, projects, education, experience, achievements, and certifications in a clean and responsive interface.

The project was built using React.js with reusable components and responsive styling.

---

## Features

- Responsive user interface
- Project showcase
- Technical skills section
- Education and experience sections
- Achievements and certifications
- Resume viewing and download functionality
- Smooth navigation
- Reusable React components

---

## Tech Stack

- React.js
- JavaScript
- CSS
- Vite

## Getting Started

### Clone the repository

```bash
git clone https://github.com/manjunath397/portfolio.git
```

### Navigate to the project

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
